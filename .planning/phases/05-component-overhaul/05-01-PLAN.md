<task type="auto">
  <name>Task 1: Rebuild Hero.tsx with Swiss asymmetric typography</name>
  <files>components/sections/Hero.tsx</files>
  <action>
Completely rewrite Hero.tsx with an asymmetric Swiss modernist layout. The hero should feel like a typographic poster, not a centered landing page.

**Layout structure (desktop):**
- Use a two-column asymmetric grid: `grid grid-cols-1 lg:grid-cols-12 gap-8 lg:gap-16 items-end`
- Left column spans 7 cols (`lg:col-span-7`): Contains the name in massive type and a brief role description
- Right column spans 5 cols (`lg:col-span-5`): Contains metrics, status indicator, and CTAs
- On mobile, stack vertically with left-alignment (NOT centered)

**Typography (left column):**
- Remove the subtitle line above the name entirely
- Name (`t('title')`) in `text-6xl md:text-7xl lg:text-[6rem] xl:text-[7.5rem] font-bold tracking-tight leading-[0.9] font-heading`
- Use plain `text-foreground` color -- NO gradient text, NO `bg-clip-text`, NO `WebkitTextFillColor`
- Below the name: `t('titleAccent')` as a secondary line in `text-3xl md:text-4xl lg:text-5xl font-heading font-medium` with `style={{ color: 'var(--text-secondary)' }}`
- Below that: `t('description')` in `text-lg` with `style={{ color: 'var(--text-secondary)' }}`, max-w-xl, left-aligned

**Right column (desktop only, stacks below on mobile):**
- Status badge: simple inline-flex with a green dot (keep the ping animation but make it subtle -- use `opacity-50` on the ping). Use `bg-[var(--color-accent-muted)]` background instead of `glass`. Remove `backdrop-blur-xl`.
- Metrics in a vertical stack (not horizontal row): each metric has the value in `font-mono text-4xl font-bold` with `style={{ color: 'var(--color-accent)' }}` and label below in `text-sm` secondary color. Space with `space-y-6`.
- CTAs: Primary button uses `bg-[var(--color-accent)] hover:bg-[var(--color-accent-hover)] text-white` with `rounded-lg` (not rounded-2xl). Secondary button uses `border border-[var(--color-border-strong)]` with `rounded-lg`, no `glass` class.

**Remove entirely:**
- The `apple-gradient-mesh` background div
- The gradient overlay div (`bg-gradient-to-b from-transparent...`)
- The bounce scroll indicator at the bottom
- All `glass` class usage
- All gradient text (`bg-gradient-to-r from-primary-blue...`, `var(--gradient-hero)`)
- The `ValuePropPill` component and value props section
- The `hover:scale-105` on buttons (Swiss design is restrained)

**Keep:**
- The `isVisible` fade-in animation (but reduce duration to `duration-500` and use `translate-y-4` instead of `translate-y-10`)
- The `useTranslations('hero')` hook and all i18n keys
- The section id (none currently, but the section should still be the first visible section)
- `MetricItem` sub-component (refactored for vertical layout and font-mono values)

**Section container:**
- `min-h-[90vh]` instead of `min-h-screen` -- leave breathing room
- `flex items-end` to push content toward the bottom of the viewport (Swiss poster feel)
- `pb-24 pt-32` for vertical rhythm on 8px grid
- `px-6 lg:px-8 max-w-7xl mx-auto` container

**Spacing:** All padding/gap values must be multiples of 8px (0.5rem = 8px, 1rem = 16px, 1.5rem = 24px, 2rem = 32px, 3rem = 48px, 4rem = 64px, 6rem = 96px).

**Navigation.tsx rewrite:**

Remove the glass-on-scroll effect entirely. The nav should use a clean solid surface when scrolled.

**Unscrolled state:** Replace `bg-transparent py-6` with `bg-transparent py-4`.

**Logo/brand mark:** Replace the gradient text logo with clean foreground text:
- Remove the inline style with `var(--gradient-hero)` and `WebkitBackgroundClip`
- Use `className="text-xl font-heading font-bold text-foreground hover:text-[var(--color-accent)] transition-colors"` instead

**Desktop nav links:**
- Keep the underline hover animation (the green `accent-500` underline is good Swiss design)
- Change the link gap from `gap-4` to `gap-6` for more breathing room (48px = 6 * 8px)

**Hire Me button:**
- Replace `rounded-full` with `rounded-lg`
- Keep `bg-[var(--color-accent)]` (the primary-600 alias resolves to this already)

**Mobile menu:**
- Replace `glass rounded-2xl` with `bg-[var(--color-surface-elevated)] border border-[var(--color-border)] rounded-xl`
- Keep all nav links, SearchBar, LanguageToggle, ThemeToggle, and Hire Me button
- Keep the hamburger animation

**Mobile overlay:** Keep `bg-black/40 backdrop-blur-sm` for the mobile overlay -- this is a functional scrim for a modal, not decorative glass. This is acceptable per the Swiss design (blur for dismissal context, not decoration).

**Do NOT change:** The `navLinks` array, keyboard shortcuts, `usePathname` logic, `isHomePage` detection, or any import. Keep SearchBar, LanguageToggle, ThemeToggle imports and placement.
