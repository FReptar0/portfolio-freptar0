  <action>
**Create `hooks/useScrollReveal.ts`:**

Create a new `hooks/` directory and file. The hook should:

1. Accept options: `{ threshold?: number; rootMargin?: string; staggerDelay?: number; once?: boolean }`
   - `threshold` defaults to 0.1 (trigger when 10% visible)
   - `staggerDelay` defaults to 0 (no stagger -- caller controls stagger via CSS custom property)
   - `once` defaults to true (only animate in once, do not re-animate on scroll back up)

2. Return `{ ref: RefCallback<HTMLElement>, isVisible: boolean }`
   - `ref` is a callback ref that attaches/detaches the Intersection Observer
   - `isVisible` becomes true when element enters viewport

3. Check `prefers-reduced-motion` at hook initialization:
   - Use `window.matchMedia('(prefers-reduced-motion: reduce)')`
   - If reduced motion is preferred, immediately set `isVisible = true` (skip animation, show content instantly)
   - Also listen for changes to the media query (user might toggle mid-session)

4. Use `IntersectionObserver` API (no libraries):
   - Create observer with provided threshold/rootMargin
   - When element intersects, set `isVisible = true`
   - If `once` is true (default), disconnect observer after first intersection

5. Clean up observer on unmount

6. Export the hook as a named export: `export function useScrollReveal(options?)`

Also create a convenience hook `useStaggerChildren` that returns a helper:
```typescript
export function useStaggerChildren(baseDelay: number = 40) {
  return {
    getDelay: (index: number) => ({
      transitionDelay: `${index * baseDelay}ms`
    })
  };
}
```

**Update `app/globals.css`:**

Add a `prefers-reduced-motion` media query block BEFORE the `@layer base` section (around line 660, after the logoloop styles):

```css
/* === Reduced Motion === */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

Also add scroll-reveal utility classes after the section-divider styles (around line 470):

```css
/* === Scroll Reveal Animations === */
.scroll-reveal {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 500ms ease-out, transform 500ms ease-out;
}

.scroll-reveal.is-visible {
  opacity: 1;
  transform: translateY(0);
}

.scroll-reveal-fade {
  opacity: 0;
  transition: opacity 500ms ease-out;
}

.scroll-reveal-fade.is-visible {
  opacity: 1;
}
```

These CSS classes are an alternative to inline styles -- components can use either the hook's `isVisible` boolean with Tailwind classes OR these utility classes. The hook approach is preferred for React components.
